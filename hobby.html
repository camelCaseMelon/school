Visual Sudoku

Visual Sudoku is a type of logic puzzle that goes by many names, including visual sudoku, which I came up with just now.



The numbers on the sides are used to create a picture- for example, if it says 13.1, there are thirteen colored squares, at least one space, and then one colored square in that row/column.

Here are some websites that let you do these puzzles online:

Web PBN, Any Puzzle

Other names for visual sudoku:

  • Griddlers
  • Nonograms
  • Picross
  • Hanjie
  • Paint by number
  • O'Ekaki

How to do visual sudoku:

  1. Fill in any rows/columns whose numbers and number of spaces between add to the total length of the row/column.
  2. If that gives you any clues (for example, a row with only one filled in square now has the square), you can now act on those!
  3. Continue until all the squares are filled and you have a picture!
